# GTNH-FTI-Faithful-Resourcepack

## RUS

### Установка

Устанавливается как обычный ресурспак для Minecraft 1.7.10.

### Новые кириллические шрифтов

Для отображения кириллического шрифта в стиле ASCII Minecraft 1.7.10 отдельные моды не требуются так как добавлена поддержка со стороны [Angelica](https://github.com/GTNewHorizons/Angelica). Связи с этим был создан отдельный ресурспак только со шрифтами [GTNH-FTI-Standard-Font](https://github.com/Eldrinn-Elantey/GTNH-FTI-Standard-Font).

## ENG

### Install

Installed as a regular resourcepack for Minecraft 1.7.10.

### To display new Cyrillic fonts

No separate mods are required to display Cyrillic fonts in ASCII Minecraft 1.7.10 style, as support has been added by [Angelica](https://github.com/GTNewHorizons/Angelica). In connection with this, a separate resource pack was created with only fonts [GTNH-FTI-Standard-Font](https://github.com/Eldrinn-Elantey/GTNH-FTI-Standard-Font).
